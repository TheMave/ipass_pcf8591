\doxysubsection{Detailed Description}
\hypertarget{README.md_autotoc_md1}{}\doxysubsection{\texorpdfstring{Inleiding}{Inleiding}}\label{README.md_autotoc_md1}
Deze folder bevat declaraties voor abstracte intefaces van een aantal typen AD en DA converters. Door de functionaliteit van converter-\/chips via deze interfaces toegankelijk te maken, kun je ze later eenvoudig vervangen door andere chips, zonder dat je de code van je applicatie hoeft aan te passen.\hypertarget{README.md_autotoc_md2}{}\doxysubsection{\texorpdfstring{Voorbeeld}{Voorbeeld}}\label{README.md_autotoc_md2}
