Keuze voor het mappen van AIN-\/signalen naar outputkanalen. 

Dit enum definieert verschillende manieren om de vier analoge inputs (AIN0–\+AIN3) om te zetten naar maximaal vier kanalen.

Mogelijke waarden\+:
\begin{DoxyItemize}
\item {\bfseries{AIN\+\_\+\+N\+\_\+\+TO\+\_\+\+CHANNEL\+\_\+N}} ~\newline
 Elke AIN\textsubscript{i} → kanaal {\itshape .}
\item {\itshape {\bfseries{AIN0\+\_\+1\+\_\+2\+\_\+\+MINUS\+\_\+\+AIN3\+\_\+\+TO\+\_\+\+CHANNEL\+\_\+0\+\_\+1\+\_\+2}} ~\newline
 (AIN0, AIN1, AIN2) − AIN3 → kanalen 0, 1, 2.}
\item {\itshape {\bfseries{AIN0\+\_\+1\+\_\+\+TO\+\_\+\+CHANNEL\+\_\+0\+\_\+1\+\_\+\+\_\+\+AIN2\+\_\+\+MINUS\+\_\+\+AIN3\+\_\+\+TO\+\_\+\+CHANNEL\+\_\+2}} ~\newline
 AIN0 → kanaal 0, AIN1 → kanaal 1, (AIN2 − AIN3) → kanaal 2.}
\item {\itshape {\bfseries{AIN1\+\_\+\+MINUS\+\_\+\+AIN0\+\_\+\+TO\+\_\+\+CHANNEL\+\_\+0\+\_\+\+\_\+\+AIN2\+\_\+\+MINUS\+\_\+\+AIN3\+\_\+\+\_\+\+TO\+\_\+\+CHANNEL\+\_\+1}} ~\newline
