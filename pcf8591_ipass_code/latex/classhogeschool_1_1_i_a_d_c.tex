\doxysubsection{Detailed Description}
Abstracte interface voor het uitlezen van ADC-\/inputs. 

Abstracte interface voor het uitlezen van differentiele ADC-\/inputs.

\begin{DoxyNote}{Note}
Een differentiele ADC-\/input bestaat uit een paar van 2 ADC inputs. De eerste tussen pinnen AIN0 en AIN1, de volgende tussen AIN2 en AIN3, etc. De uitgelezen waarde is het verschil tussen de eerste en de tweede input van het paar. Dat verschil kan negatief zijn. 
\end{DoxyNote}
